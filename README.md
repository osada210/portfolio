@anime　というコマンドを送信するとアニメ情報が複数のフレックスメッセージ(カルーセル形式)で返ってくるbotです

横、縦にスクロールすることで最新のアニメ情報すべてを見ることができます

スクレイピングを行う関係上、トーク相手が増えすぎるとリクエスト先に負荷がかかる可能性があるため、QRコード等での友達追加は受け付けていません

将来的にコマンドを追加し、過去のアニメデータの取得などの実装も行う予定です
